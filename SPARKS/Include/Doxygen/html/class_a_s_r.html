Detailed Description

This class encapsulates the Loquendo speech recognition engine.

At initialization time the class builds an on-the-fly recognition object that is stored to disk, starting with a plain text big grammar file.
Voice recognition is done by specifying a certain rule of that grammar, that becomes a virtual root rule of the grammar, thus implimenting state dependent grammars.

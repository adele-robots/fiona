class ComponentSystem

Detailed Description

All components derive from this class. They are forced to implement initializeRequiredInterfaces(). The component loader instantiates the components, creates the interface repository, and calls the initializeRequiredInterfaces() of all instantiated components, thus filling the required interface pointers.

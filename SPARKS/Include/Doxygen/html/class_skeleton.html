Detailed Description

This class for joint manipulation.

Skeletons have joints which are scene nodes of which other joints are daughters in the scene graph. Rotating a joint causes the corresponding deformation of the associated body mesh. Thus joint rotation angles are the elemental degrees of freedom of the character kinematics.

Currently joint translations are not used with the possible exception of the root joint to translate the whole character.

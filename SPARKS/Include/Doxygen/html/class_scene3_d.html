Detailed Description

This class encapsulates the 3D scene rendering implementation.

Scenes are 3D, time variying geometries. To render them we use a 3D enging. This class exploits the engine facilities. The main structure is the Scene Tree. It has a root node. The rest of the nodes have transformations, (traslations, rotations and scale). They are compounded as one traverses the Tree. Other scene elements are meshes, textures, camara and lights.
